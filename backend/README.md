## Folder Structure
- [Config](./config):
  Environment configuration for the database
- [Data](./data):
  A generator for mock data in json form and default data in csv form.
- [Models](./models):
  Sequelize model definitions
- [Services](./services):
  Backend api
- [Utils](./utils):
  Utility functions for parsing csv and manipulating default data
